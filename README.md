# SHIELD Project (URES Spring/Summer 2025)

This is all of the code I have used for URES in Spring & Summer 2025 working on the SHIELD project in Pamela Martinez's lab. 

This repository includes:

### Tripledemic Paper
- tripledemic_model

    - beta_vals.Rmd --> Code to find beta values from target R0
    - tripledemic.Rmd --> Code to run SEIR & build figures
